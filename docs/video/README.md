# AI Coding Assistant Demo Video Pack

This folder contains everything needed to record and share a short demo video about how AI coding assistance was used to build this project.

## Files

- `VIDEO_SCRIPT.md` — full narration script
- `STORYBOARD.md` — timeline + what to show on screen
- `captions.srt` — subtitles for upload platforms

## Fast Recording Flow (10–15 minutes)

1. Start the app:
   - `npm run dev --prefix server`
   - `npm run dev --prefix client`
2. Open your screen recorder (Loom/OBS/Zoom local recording).
3. Follow `STORYBOARD.md` while reading `VIDEO_SCRIPT.md`.
4. Upload your video to YouTube (unlisted) or Loom and share the link in your applications.
